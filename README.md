# CH4-Cr2N-DFT

> **Theoretical Evaluation of the Catalytic Potential of Chromium Nitride for Methane Activation via First-Principles Calculations**

This repository contains the computational workflow, input files, analysis scripts, and processed data used in the study presented at the University of Costa Rica.

## Overview

Methane (CH₄) is the primary component of natural gas and an attractive feedstock for the production of fuels and value-added chemicals. However, the remarkable stability of its C–H bonds makes methane activation one of the major challenges in heterogeneous catalysis.

This project investigates the interaction between methane and the Cr-terminated Cr₂N surface using **Density Functional Theory (DFT)** calculations performed with **Quantum ESPRESSO**. Twenty-seven initial adsorption configurations were generated by combining different methane orientations, rotations, and adsorption sites. After geometry optimization, the six most stable configurations were selected for detailed structural and electronic analysis.

The repository provides a reproducible workflow from the generation of adsorption structures to the analyses presented in the accompanying academic poster.

# Workflow

## 1. Initial adsorption configurations

The notebook

```
notebooks/00_generate_initial_configs.ipynb
```

generates the complete set of **27 initial CH₄ adsorption configurations** by combining

- three methane orientations
- three molecular rotations
- three adsorption sites

using the reference structures

```
structures/slab.xyz
structures/CH4.xyz
```

The generated configurations are **not included** in the repository because they can be reproduced directly from the notebook.

---

## 2. Quantum ESPRESSO input generation

The Python script

```
scripts/01_generate_qe_inputs.py
```

automatically generates Quantum ESPRESSO input files for all adsorption configurations.

---

## 3. Geometry optimization

Geometry optimizations were performed using **Quantum ESPRESSO 7.5**.

The repository contains

- relaxation input files
- output files
- relaxed geometries

for the complete set of adsorption configurations.

---

## 4. Structural analysis

The analysis notebooks compute

- adsorption energies
- Cr surface displacements
- C–H bond elongation

used to produce the figures shown in the poster.

---

## 5. Electronic structure

Electronic analyses include

- SCF calculations
- NSCF calculations
- Projected Density of States (PDOS)
- Charge-density difference (Δρ)

# Computational Details

Software

- Quantum ESPRESSO 7.5
- ASE
- Python
- NumPy
- Matplotlib

Exchange-correlation functional

- PBE-GGA

Dispersion correction

- DFT-D3

Plane-wave cutoff

```
ecutwfc = 70 Ry
ecutrho = 840 Ry
```

k-point mesh

```
4 × 4 × 1
```

Smearing

```
Methfessel–Paxton
degauss = 0.02 Ry
```
